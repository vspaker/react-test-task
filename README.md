## Тестовое задание на позицию Junior Frontend Developer (приложение на React)

Для запуска проекта:

* `npm install`
* `npm install --save prop-types`
* `npm i --save lodash`
* `npm install react-paginate --save`
* `npm start`

После этого приложение будет доступно по адресу `http://localhost:3000/`
